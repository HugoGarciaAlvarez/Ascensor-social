## Instalacion
### 1.Requisitos
- Node.js 18
- npm(incluido en Node)
## 2. Clonar el proyecto
```bash
git clone https://github.com/HugoGarciaAlvarez/Ascensor-social.git

```

---

## 3. Arrancar el backend (API con Node.js)

```bash
cd server
npm install
npm run dev
```

API en: [http://localhost:5174/api/health](http://localhost:5174/api/health)

---

## 4. Arrancar el frontend (Svelte)

```bash
cd ../client
npm install
npm run dev
```

Dashboard en: [http://localhost:5173](http://localhost:5173)
